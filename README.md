# Symfony3SimpleExample
Symfony3 With DataTable Simple Example


1 - Install app
 ``` bash
composer install
```
2 - www run
``` yml
 {your.domain}/app_dev.php/user/table
 ```
 3- command run
``` bash
   php bin/console random-user
```
or with --ftable option
``` bash
 php bin/console random-user -f
```
